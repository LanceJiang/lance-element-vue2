# admin-ui
vue-cli3构建基于elementUI二次封装库 并集成 vxe-table

### Install
```
npm install @lancegroup/admin-ui -S
```

### Usage
```
import Vue from "vue"
import AdminUI from '@lancegroup/admin-ui'
import '@lancegroup/admin-ui/style/index.scss'
Vue.use(AdminUI)
```
在项目中使用
```
<ad-component></ad-component>
```
